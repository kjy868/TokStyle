토마코의 할로윈! ( ios/and )

디도리 2025.10.30

할로윈을 맞아 토마코는 유령으로, 토마토는 잭오랜턴으로 변신했어요!
모두 으스스한..아니 귀여운 할로윈 되시길~

디도리 일상에 단정한 귀여움을!

테마 상세

이번 테마는 작고 무서운(?) 친구들과 함께 보내는 [토마코의 할로윈!]입니다.

사용한 폰트: '도스샘물'

*메세지 입력창이 흰색인 경우(버전1),
#검색 시 자동완성 검색어 글씨가 보이지 않는 점이 확인되어
메세지 입력창이 어두운 버전2를 추가로 제작했습니다.
원하는 버전으로 다운받아 사용해 주세요!

설치 방법

iOS

  1. 파일 다운로드 다운로드 바로가기를 눌러 원하는 테마를 다운로드 받아주세요.
    *반드시 “ktheme” 확장자를 확인하셔야 합니다.
  2. 카카오톡 나에게 공유하기 다운로드한 테마 파일을 ‘공유하기’ 버튼으로 카카오톡에서 ‘나와의 채팅(나에게)’으로 전송합니다.
    *PC에서 받은 파일은 메일이나 카카오톡 등으로 내 아이폰에 옮겨도 괜찮습니다.
  3. 공유한 파일 누르고 테마 적용하기 카카오톡 앱을 열어 전송된 .ktheme 파일을 누르면
    나오는 ‘테마 적용하기’ 메뉴를 선택합니다.

Android

  1. 파일 다운로드 다운로드 바로가기를 눌러 원하는 테마를 다운로드 받아주세요.
    *반드시 “apk” 확장자를 확인하셔야 합니다.
  2. 앱 설치하기 다운로드가 완료되면 파일을 눌러주세요.
    *출처를 알 수 없는 앱 경고문이 뜨면 무시하고 설치를 눌러주셔야 합니다.
  3. 설치된 앱 들어가서 테마 적용하기 설치된 앱을 열면 나오는 ‘테마 적용하기’ 버튼을 누르면 테마가 적용됩니다.
